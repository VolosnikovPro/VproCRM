# VproCRM

CRM-система для управления клиентами с интеграцией Avito Messenger.

## Возможности

- **Воронка продаж (Kanban)**: настраиваемые этапы с drag-and-drop, цветовые индикаторы, суммы бюджетов
- **Карточки клиентов**: сделка, контакт, телефон, email, организация, адрес, ответственный, бюджет, источник, теги
- **Задачи**: с датой и временем, статус выполнения, группировка (просрочено/сегодня/на неделе/позже)
- **Заметки**: текст + файлы к клиенту
- **Дашборд**: статистика по этапам, источникам, задачам, бюджетам, графики
- **Календарь**: сетка задач по дням
- **Теги**: цветные, many-to-many, фильтрация
- **Поиск и фильтры**: по тексту, этапу, источнику, тегу
- **Экспорт/импорт CSV**
- **Тёмная тема**
- **Уведомления**: за 15 мин и в момент дедлайна задачи
- **Резервное копирование и восстановление**
- **Объединение дубликатов**
- **Hotkeys**: Ctrl+N, Ctrl+F, Escape, Ctrl+S
- **Лента активности**: задачи, заметки, вложения, сообщения Avito

### Интеграция с Avito

- Подключение/отключение аккаунта Avito (OAuth2 client_credentials)
- Синхронизация чатов и сообщений (SDK V2/V3)
- Автообновление чатов каждые 60 секунд
- Отправка сообщений через Avito
- Синхронизация объявлений (core/v1/items)
- **Статистика объявлений**:
  - Ежедневные показы, просмотры, контакты, избранное, расходы
  - Данные загружаются через API Avito V2 (1 запрос/мин)
  - Хранение по дням в БД, агрегация за произвольный период
  - Прогресс загрузки с отображением в UI
- Причины отказов с обязательным выбором при переносе на этап "Отказ"

## Стек

- Python 3.14, FastAPI, SQLAlchemy, SQLite
- HTML/CSS/JS (vanilla)
- PyInstaller (сборка .exe)

## Запуск

### Из исходников

```bash
pip install -r requirements.txt
python main.py
```

Откроется в браузере по адресу http://127.0.0.1:8000

### Из .exe

Запустить `dist/VproCRM.exe` — сервер стартует автоматически.

## Сборка .exe

```bash
python -m PyInstaller VproCRM.spec
```

## Структура

```
├── server.py         # API сервер (FastAPI)
├── main.py           # Точка входа
├── database.py       # Подключение к БД, миграции
├── models.py         # SQLAlchemy модели
├── schemas.py        # Pydantic схемы
├── static/           # Фронтенд
│   ├── index.html
│   ├── app.js
│   └── style.css
├── VproCRM.spec      # Конфиг PyInstaller
└── requirements.txt  # Зависимости
```

## Важно

- База данных `crm.db` создаётся автоматически рядом с .exe (или в папке проекта)
- При перезапуске данные сохраняются
- Для Avito необходимы client_id и client_secret (настройка в разделе Интеграции)
